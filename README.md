# Africa Free Routing

A simple donation page powered by Blink Bitcoin Lightning payments.

## Donate

Visit [https://pretyflaco.github.io/africafreerouting/](https://pretyflaco.github.io/africafreerouting/) to donate 100 sats via Lightning Network.

## Tech

- Static HTML/CSS/JS
- [Blink API](https://blink.sv/) for Lightning invoices
- [QRCode.js](https://cdn.jsdelivr.net/npm/qrcode@1.5.3/build/qrcode.min.js) for QR generation
